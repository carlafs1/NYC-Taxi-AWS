Consulta direta em gold.trips (tabela Iceberg de verdade, lida via iceberg_scan()), filtrada para Yellow Cab. O próprio Iceberg faz o pruning de partição pelos manifests — só os arquivos do mês escolhido são lidos, mesmo com anos de histórico acumulado.
