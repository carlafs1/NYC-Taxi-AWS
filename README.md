## 🎯 Objetivo

Construir uma plataforma de dados na AWS utilizando serviços gerenciados,
Infraestrutura como Código (IaC) e processamento distribuído.

---

## 🧰 Tecnologias

| Categoria | Serviço/Ferramenta |
|---|---|
| Armazenamento | Amazon S3 |
| Ingestão | AWS Glue (Python Shell) |
| Processamento | Amazon EMR Serverless (Apache Spark / PySpark) |
| Formato de tabela | Apache Iceberg |
| Orquestração | AWS Step Functions |
| Agendamento | Amazon EventBridge |
| Catalogação | AWS Glue Data Catalog |
| Infraestrutura como Código | Terraform |
| CI/CD | GitHub Actions |
